### Overview

This repository contains the GitHub project for a **dynamic and flexible hierarchical Gantt chart solution**.

The repository is divided into **two main parts**:

### 1. Main package
The main package provides:
- **Comprehensive documentation**
- The **core implementation**, specifically adapted for **Power BI**

### 2. "HappyFew" package
The **HappyFew** package allows you to use the same Gantt solution in **any JavaScript environment**, independently of Power BI.

---

### Features

With this Gantt chart, you can:
- Define and modify the **hierarchical structure** of the Gantt chart at any time
- Customize **colors**
- Add **custom milestones** (shape, size, and color)
- Configure the **height of task bars**
- Display **text inside task bars**
- Visually track the **progress of your tasks and projects**
